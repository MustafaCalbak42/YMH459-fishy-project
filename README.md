# Balık Müzesi 3D Modelleme Uygulaması

## Proje Hakkında
Bu proje, balık müzelerindeki balıkları 3D modelleme yapmak için geliştirilmiş Unity tabanlı bir uygulamadır.

## Proje Yapısı

```
YMH459-fishy-project/
│
├── Assets/
│   ├── Scripts/           # C# script dosyaları
│   │   ├── Modeling/      # 3D modelleme scriptleri
│   │   ├── UI/            # Kullanıcı arayüzü scriptleri
│   │   ├── Camera/        # Kamera kontrol scriptleri
│   │   └── Utils/         # Yardımcı araçlar
│   │
│   ├── Scenes/            # Unity sahne dosyaları
│   ├── Models/            # 3D model dosyaları
│   │   ├── Fish/          # Balık modelleri
│   │   └── Environment/   # Çevre modelleri
│   │
│   ├── Materials/         # Materyal dosyaları
│   ├── Textures/          # Doku dosyaları
│   ├── Prefabs/           # Prefab objeleri
│   ├── UI/                # UI elemanları
│   └── Audio/             # Ses dosyaları
│
├── ProjectSettings/       # Unity proje ayarları
├── Documentation/         # Proje dokümantasyonu
└── README.md
```

## Gereksinimler
- Unity 2021.3 LTS veya üzeri
- .NET Framework 4.7.1 veya üzeri

## Kurulum
1. Unity Hub'ı açın
2. "Open" butonuna tıklayın
3. Bu proje klasörünü seçin
4. Unity Editor'de projeyi açın

## Özellikler (Planlanmış)
- 3D balık modelleme araçları
- Etkileşimli kullanıcı arayüzü
- Kamera kontrolleri
- Model kaydetme/yükleme sistemi
- Materyal ve doku düzenleme
