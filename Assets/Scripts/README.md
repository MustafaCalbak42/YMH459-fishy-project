# Scripts Klasörü

Bu klasör Unity C# script dosyalarını içerir.

## Alt Klasörler

### Modeling/
3D modelleme ile ilgili scriptler:
- Mesh manipülasyonu
- Vertex düzenleme
- Model oluşturma araçları

### UI/
Kullanıcı arayüzü scriptleri:
- Menü kontrolleri
- Buton işlevleri
- Panel yönetimi

### Camera/
Kamera kontrol scriptleri:
- Orbit kamera
- Zoom kontrolleri
- Kamera geçişleri

### Utils/
Yardımcı araçlar:
- Dosya işlemleri
- Veri yönetimi
- Genel yardımcı fonksiyonlar
